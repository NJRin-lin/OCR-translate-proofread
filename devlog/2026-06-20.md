# 2026-06-20 — Windows 版本初始迁移

## 目标

基于 macOS SwiftUI 原项目（NGM proofread），开发 Windows WPF 版本，保持功能、视觉风格和交互逻辑一致。

## 技术选型

- **UI 框架**: WPF (.NET 8)
- **OCR**: Windows.Media.Ocr（WinRT 互操作，支持日语）
- **TTS**: System.Speech（Microsoft Haruka Desktop，ja-JP）
- **AI API**: HttpClient + System.Text.Json（DeepSeek / OpenAI / Gemini）
- **截图**: Win32 GDI+ Graphics.CopyFromScreen
- **存储**: JSON 文件（%LocalAppData%/NGMproofread/）

### 9 大功能模块

1. **图片识别** — 截图 / 上传图片（PNG/JPEG/BMP/TIFF），图片缩放/平移（1x-5x），框选裁剪（8 手柄 + 遮罩叠加层 + 光标适配）
2. **文字输入** — 侧边栏双文本编辑区，可选参考对照图片
3. **本地 OCR** — Windows.Media.Ocr，日语优先，分段/连贯视图，置信度颜色编码（绿/橙/红），TTS 朗读

5. **AI 句子分析** — 校对模式（成分树）/ 学习模式（成分树 + 语法点 + N1-N2 词汇 + 备注），按句子卡片展示，缓存复用
6. **词汇查询** — AI 词典查询，Ctrl+K 聚焦，展开/收起卡片
7. **TTS 朗读** — System.Speech，已验证系统装有日语语音 Haruka
8. **设置面板** — API Key 管理 / 模型选择 / 术语表 / 分析深度
9. **截图** — GDI+ 全屏截图

## 已解决的问题

- **NU1301**: 机器级 NuGet fallbackPackageFolders 指向不存在的路径 → `mkdir` 创建目录
- **CS0246**: HttpClient 未找到 → 添加 `using System.Net.Http`
- **CS0103**: Path/Directory/File 未找到（26 处）→ 添加 `using System.IO`
- **CS0234**: `Windows.Storage` 等被项目根命名空间 `NGMproofread.Windows` 遮盖 → `global::Windows.*` 类型别名
- **CS1061**: `Windows.Foundation.Rect` 非可空 → 直接访问 `BoundingRect`
- **CS8209**: `RunAsync` 返回 void 无法 discard → 改为 `Task`

## 运行时 Bug 修复（共 20+ 项）

### 图片裁剪交互
- **选区拖动加速度 Bug**：`MoveSelection` 每帧将累积 delta 加到已移动的位置，导致 N×delta 加速度。修复：拖动开始时保存 `_selectionStartOrig`/`_selectionEndOrig`，每帧从原始值 + 累积 delta 计算
- **清除选区后模式**：`ClearSelectionBtn_Click` 现在将 `_isSelectionMode = false`，恢复默认平移/缩放模式
- **选区确认后不自动缩放**：`ConfirmSelectionBtn_Click`、`FullImageBtn_Click`、`CancelSelectionBtn_Click` 移除 `ApplyZoom(1.0)` 调用，保持用户缩放/平移位置

## 分发

### 方案一：自包含单文件 EXE（直接运行，无需安装）
- 路径：`win\publish\NGMproofread.Windows.exe`
- 大小：~180MB（包含完整 .NET 8 运行时）
- 用法：下载后直接双击运行，无需安装任何依赖

### 方案二：安装器（推荐分发方式，文件小）
- 路径：`win\publish\NGMproofread_Setup_1.1.1.exe`
- 大小：~4MB（NSIS 安装器，lzma 固实压缩）
- 制作工具：NSIS 3.12（脚本：`win\installer\setup.nsi`）
- 发布方式：框架依赖（framework-dependent），不含运行时
- 安装流程：
  1. 运行 `NGMproofread_Setup_1.1.1.exe`
  2. 自动检测 .NET 8 桌面运行时，未安装则引导下载
  3. 安装到 `Program Files\NGMproofread`
  4. 创建桌面快捷方式 + 开始菜单项 + 卸载程序
- 运行时依赖：.NET 8 Desktop Runtime（x64）
  - 下载地址：https://dotnet.microsoft.com/download/dotnet/8.0

### 发布命令
```
# 自包含（180MB，用户无需安装 .NET）
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true -o ../publish

# 框架依赖（~26MB 原始文件，安装器压缩至 ~4MB）
dotnet publish -c Release -r win-x64 --self-contained false -o ../publish-fd
```
