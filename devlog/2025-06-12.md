# 开发日志

## 2025-06-12 ~ 2025-06-14

### 项目基础
- SwiftUI + Swift | macOS 14+ | Xcode 26.5 | DeepSeek API | Vision 框架
- 22 源文件，零警告零错误

### 核心功能
- 截图/上传 → 框选选区 → OCR（Vision 本地，不依赖 API）→ 翻译（DeepSeek）→ 分析
- 翻译分析手动触发，OCR 自动运行
- 模式切换自动触发翻译（已有 OCR 时）

### 图片框选
- 拖拽创建选区、8 手柄精调（4 角 + 4 边）、选区内部拖拽移动
- SF Symbol 对角光标 + 十字星 + 手掌拖动
- 缩放 100%~500%：滚轮 + 触控板捏合 + 按钮，累计连续
- 平移不限缩放，80px 边界保留，触控板滑动 vs 鼠标滚轮自动识别
- 百分比坐标映射裁剪，避免累积误差
- 滚动事件仅响应图片区域（FrameReader 命中检测），Sheet 弹出时跳过

### 分析模式
- **校对模式**：精准句子成分拆解，无语法/词汇，Markdown 表格展示
- **学习模式**：句子成分 + 语法点（附例句）+ 词汇注解（仅 N1~N2）
- 模式缓存，切换模式即时显示
- 工具栏内嵌 linear 进度条

### 词汇查询
- 查词输入框 + 蓝色查询按钮，支持 Cmd+K 快速聚焦
- 浮动卡片展示结果（读音/释义/词性/JLPT/例句），折叠/展开
- 查询缓存
- 查词栏固定在翻译和分析之间，滚动时始终可见

### 翻译结果
- 原文 + 中文双区展示，原文去除换行空格
- 原文可选中复制
- 句子分析原文可选中

### UI 细节
- 三栏布局：图片 | OCR 识别 | 翻译+分析
- 初始界面「上传图片」蓝色高亮
- 工具栏固定高度，文字单行不换行
- 缩放按钮：1:1 | − | % | +
- API Key 用 UserDefaults 存储
- 光标样式：角/边/内部/外部区分
